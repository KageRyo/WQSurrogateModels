# Legacy Training Scripts

This directory contains the older exploratory training scripts that previously
lived under `src/training/`.

They are kept for historical traceability only. Current reproducibility and
training workflows live under `scripts/`, including:

- `scripts/reproduce_results.py`
- `scripts/run_sample_size_experiments.py`
- `scripts/run_missing_indicator_experiments.py`
- `scripts/run_missing_indicator_robustness_experiments.py`

Do not use these archived scripts as the source of current manuscript results or
production model artifacts.
